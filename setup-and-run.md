# Setup and Run

## Install
```bash
python3 -m pip install -r requirements.txt
```

## Launch App
```bash
PYTHONPATH=. streamlit run app.py
```

## Run CLI Pipeline
```bash
PYTHONPATH=. python3 scripts/run_pipeline.py <path_to_csv>
```

## Sample File
Use `sample_data/sample_bank_statement.csv` for a quick demo.
